# Nursing School — Study Guides

Verbatim Markdown study guides extracted from Chamberlain University Edapt
homework modules, organized by course and week.

## Courses

| Folder | Course | Coverage |
| --- | --- | --- |
| [`maternal_child/`](maternal_child/) | Maternal-Child Nursing | Weeks 4–7 + an NCLEX practice quiz |
| [`nutrition_nr228/`](nutrition_nr228/) | NR228 Nutrition | Weeks 4–7 |

## Structure

- One `.md` file per module, named `Week{N}_{Module_Name}.md`.
- Each file captures the Introduction and every Explore section verbatim
  (tabs, carousel slides, accordions, and transcripts expanded), and ends with a
  completion audit listing what was captured and skipped.
- Prepare, Self-Check, and Reflect sections are intentionally omitted.
- Diagrams worth keeping live under each course's `assets/` folder.

*Content is coursework belonging to Chamberlain University / Edapt, captured here
for personal study reference.*
